AMC - Requiring an Alternate

Interactive TAF practice for destination alternate decisions
CAO 29 MAY 26
Example 1

Landing at 28 1900Z. Which line applies, and do we need an alternate?

KMCF TAF Tap or hover any boxed part for the decode.
Line selection

The ETA is at the start of the BECMG window and the ±1 hour window captures the change. BECMG becomes effective by the end time, so this is the timing trap.

Decision

Alternate required. The forecast includes a ceiling below 2,000 feet AGL and visibility below 3 statute miles within the required window. (202V3, 4.16.2.1)

Two-alternate filing note

File two alternates when forecast visibility is less than published for the available precision approach. Also file two alternates when forecast ceiling or visibility is less than published for all other approaches.

Example 2

Landing at 29 0200Z. Which line applies, and do we need an alternate?

KSKA TAF Tap or hover any boxed part for the decode.
Line selection

The BECMG line is effective by 0200Z, so this line applies at the ETA.

Decision

Alternate required if the forecast winds are outside aircraft limitations. (202V3, 4.16.2.3)

Example 3

This line applies to the landing. Do we need an alternate?

ETAR TAF Tap or hover any boxed part for the decode.
Weather read

The weather itself looks excellent: light winds, unrestricted visibility, and clear sky.

Decision

Alternate required unless an AMC exception applies. The trigger is not bad weather; it is the OCONUS destination rule. (202V3, 4.16.6.1)
